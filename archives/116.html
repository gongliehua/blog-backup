Elasticsearch 安装

Elasticsearch 简称 Es，提供搜索服务的一个软件

https://www.elastic.co/cn/elasticsearch/

安装：
先去下载安装包，linux也能通过yum/apt命令安装，具体安装方法看官网，我这边使用的使用时ubuntu，采用下载安装包方式安装。
下载后通过命令 dpkg -i xxxx.deb 安装，使用 ststemctl start elasticsearch 命令启动。
配置文件在 /etc/elasticsearch/ 目录下。

文档
官网中文文档只有2.x版本的，最新版本中文文档在 learnku社区有
https://learnku.com/docs/elasticsearch73/7.3

跑起来的时候，我想使用的是虚拟机，想要宿主机也能访问，于是修改了network.host，
此处参数可以写_local_/_site_/_global_
参考文档 https://www.elastic.co/guide/en/elasticsearch/reference/7.3/modules-network.html#network-interface-values

结果重启不了，经过查看日志(/var/log/elasticsearch/elasticsearch.log) ERROR 得知 node.name validation exception 。上网搜索 得到解决 方法是 把 node.name，cluster.initial_master_nodes 注释打开（cluster.initial_master_nodes 把node-2删除）。

顺利启动。
